## 4. 指标口径说明

用户题目中写的是 `ART`。技术分析中更常见的是 `ATR`，即 Average True Range，平均真实波幅。本 spec 暂按 `ATR` 处理；如果用户确认确实需要其他名为 ART 的指标，再调整公式。

### 4.1 RSI

中文名：相对强弱指标。

默认参数：

- `RSI_6`
- `RSI_14`
- `RSI_24`

计算逻辑：

- 计算收盘价变化。
- 将上涨和下跌分别平滑。
- `RSI = 100 - 100 / (1 + RS)`

解读重点：

- RSI 高于 70：偏超买。
- RSI 低于 30：偏超卖。
- 对强趋势股票，超买不一定立即卖出，需结合趋势和成交量确认。

### 4.2 MACD

中文名：指数平滑异同移动平均线。

默认参数：

- 快线 EMA：12 日
- 慢线 EMA：26 日
- Signal：9 日

输出字段：

- `macd_dif`
- `macd_dea`
- `macd_hist`

解读重点：

- DIF 上穿 DEA：偏多信号。
- DIF 下穿 DEA：偏空信号。
- 柱状图放大代表趋势动能增强，收缩代表动能减弱。

### 4.3 Bollinger Bands

中文名：布林带。

默认参数：

- 均线窗口：20 日
- 标准差倍数：2

输出字段：

- `bb_mid`
- `bb_upper`
- `bb_lower`
- `bb_width`
- `bb_percent_b`

解读重点：

- 价格靠近上轨：趋势强或短期过热。
- 价格靠近下轨：趋势弱或短期超跌。
- 布林带宽度扩大：波动加剧。
- 布林带宽度收窄：可能进入波动压缩阶段。

### 4.4 ATR

中文名：平均真实波幅。

默认参数：

- 14 日。

计算逻辑：

- `TR = max(high - low, abs(high - pre_close), abs(low - pre_close))`
- `ATR = TR` 的 14 日滚动均值或 Wilder 平滑。

执行时建议使用 Wilder 平滑，因为它是更常用的 ATR 口径。

解读重点：

- ATR 上升：市场波动和交易风险增加。
- ATR 下降：市场波动收敛。
- ATR 可用于止损距离、仓位管理和波动率分层。

### 4.5 OBV

中文名：能量潮。

计算逻辑：

- 若今日收盘价高于昨日收盘价，则 OBV 增加今日成交量。
- 若今日收盘价低于昨日收盘价，则 OBV 减少今日成交量。
- 若持平，则 OBV 不变。

解读重点：

- 价格上涨且 OBV 同步上升：上涨有成交量支持。
- 价格上涨但 OBV 背离：上涨可靠性下降。
- OBV 更适合观察方向和背离，不适合直接比较 A 股与港股绝对值。

### 4.6 ADX

中文名：平均趋向指数。

默认参数：

- 14 日。

输出字段：

- `plus_di`
- `minus_di`
- `adx`

解读重点：

- ADX 高于 25：趋势强度较高。
- ADX 低于 20：趋势较弱或震荡。
- `+DI` 高于 `-DI` 且 ADX 上升：多头趋势更强。
- `-DI` 高于 `+DI` 且 ADX 上升：空头趋势更强。

### 4.7 BIAS

中文名：乖离率。

默认参数：

- `BIAS_6`
- `BIAS_12`
- `BIAS_24`

计算逻辑：

- `BIAS_N = (close - MA_N) / MA_N * 100`

解读重点：

- 正乖离过大：短期偏离均值过高，可能有回撤压力。
- 负乖离过大：短期偏离均值过低，可能有修复需求。
- 对趋势强的股票，乖离率应结合趋势强度和成交量判断。

### 4.8 Z-score

中文名：标准分数。

默认参数：

- 20 日滚动窗口。

计算逻辑：

- `zscore_20 = (close - rolling_mean_20) / rolling_std_20`

解读重点：

- Z-score 高于 2：价格显著高于近期均值。
- Z-score 低于 -2：价格显著低于近期均值。
- Z-score 适合均值回复分析，但在趋势行情中可能连续处于极端区间。

## 5. 可视化设计

HTML 报告应支持 A 股和港股切换，建议使用页面顶部的分段按钮或 tab：

- `A股 688981.SH`
- `港股 00981.HK`

每个市场包含以下图表，每张图都需要编号、标题、图例和解读。

### 图 1：价格与布林带

内容：

- 收盘价。
- 布林带上轨、中轨、下轨。

标题示例：

- `图1 A股 688981.SH 收盘价与布林带`
- `图1 港股 00981.HK 收盘价与布林带`

解读重点：

- 当前价格处于布林带中的位置。
- 是否出现带宽扩张或收缩。
- 是否存在突破上轨、跌破下轨后回归的现象。

### 图 2：RSI

内容：

- RSI_6、RSI_14、RSI_24。
- 70 和 30 参考线。

解读重点：

- 是否出现超买/超卖。
- 短周期 RSI 与长周期 RSI 是否背离。

### 图 3：MACD

内容：

- DIF。
- DEA。
- MACD 柱状图。

解读重点：

- 金叉、死叉。
- 柱状图扩张或收缩。
- 趋势动能是否增强或减弱。

### 图 4：ATR

内容：

- ATR_14。
- 可选叠加 `ATR_14 / close * 100`，用于跨市场比较相对波动。

解读重点：

- 波动风险是否上升。
- 后续策略止损和仓位是否需要动态调整。

### 图 5：OBV

内容：

- OBV 曲线。
- 可选叠加 OBV 移动平均线。

解读重点：

- 量价是否同步。
- 是否存在价格创新高但 OBV 未创新高的背离。

### 图 6：ADX 与 DI

内容：

- ADX。
- +DI。
- -DI。
- 20 和 25 参考线。

解读重点：

- 趋势强度是否足够。
- 多空方向是否清晰。

### 图 7：BIAS

内容：

- BIAS_6。
- BIAS_12。
- BIAS_24。
- 0 轴参考线。

解读重点：

- 价格偏离均线的程度。
- 是否进入短期过热或超跌区域。

### 图 8：Z-score

内容：

- Z-score_20。
- +2、0、-2 参考线。

解读重点：

- 价格相对近期均值是否处于极端位置。
- 对均值回复或趋势延续策略的启示。
