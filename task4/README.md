# TASK4 海龟交易策略回测看板

这是一个单标的海龟交易策略回测看板，支持在浏览器中导入日线 CSV、检查数据质量、判断是否适合海龟策略，并展示唐奇安通道突破、ATR 止损、金字塔加仓、反向突破离场、净值曲线、交易信号和交易明细。

## 主要功能

- 数据导入与质量检测：检查日期范围、核心 OHLCV 缺失、重复日期、价格关系异常、成交量异常，并提示是否疑似包含复权信息。
- 海龟适配初判：根据样本长度、数据质量和波动水平，给出是否适合海龟策略的初步判断与原因。
- 突破入场系统：盘中价格突破前 `N` 日唐奇安上轨后，下一交易日开盘买入。
- ATR 系统：使用 `N` 日 ATR 计算单位头寸、动态止损和加仓间隔。
- 单位头寸计算：`可买股数 = 账户资金 x 1% / ATR`，再按每手股数向下取整。
- ATR 动态止损：`止损价 = 最近一次入场或加仓成交价 - 2 x ATR`。
- 加仓管理：初始入场 1 个单位，价格每上涨 `0.5 x ATR` 加仓 1 个单位，最多持有 4 个单位。
- 止盈离场：持仓后盘中价格跌破前 `N` 日最低价，触发反向突破离场信号。
- 风险控制：单标的强制最多 4 个单位，并在参数快照中说明标准海龟组合级风险规则。
- 交易成本：支持手续费、最低手续费、卖出税费和滑点模拟。
- 图表交互：展示收盘价、唐奇安高低通道、ATR 止损线、买入/加仓/卖出信号、净值、基准、持仓单位和回撤。
- 报告导出：支持导出 HTML 报告，也可通过浏览器打印为 PDF。

## 快速开始

直接用浏览器打开：

```text
task4/turtle_strategy_dashboard.html
```

如果浏览器禁止 `file://` 页面自动读取示例 CSV，可以手动导入：

```text
task4/sample_data/smic_a_daily_qfq.csv
```

也可以通过本地静态服务器打开，页面会自动尝试加载默认示例数据：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000/task4/turtle_strategy_dashboard.html
```

## 默认参数

- 初始资金：`100000`
- 入场唐奇安通道：`20`
- ATR 周期：`20`
- 单位风险比例：`1%`
- ATR 止损倍数：`2`
- 加仓间隔：`0.5 x ATR`
- 最多单位：`4`
- 反向突破离场：`10`
- 无风险年化收益率：`2%`

## 命令行复现

运行中芯国际 A 股示例：

```bash
python3 -B task4/turtle_backtest_engine.py task4/sample_data/smic_a_daily_qfq.csv --out-dir task4/exports_smic_default
```

运行比亚迪 A 股示例：

```bash
python3 -B task4/turtle_backtest_engine.py task4/sample_data/byd_a_daily_qfq.csv --out-dir task4/exports_byd_default
```

命令行会输出：

- `turtle_summary.json`
- `turtle_trades.csv`
- `turtle_signals.csv`
- `turtle_equity.csv`

这些输出目录已被 `.gitignore` 排除，可以按需本地重新生成。

## 回测口径

- 信号在第 `t` 日收盘后产生。
- 订单在第 `t+1` 个交易日开盘价成交；遇到周末或休市日会顺延到下一个有数据的交易日。
- 唐奇安通道使用信号日前已经完成的 `N` 个交易日，不使用当日高低价计算阈值。
- 入场、加仓、ATR 止损和反向突破离场均使用日内高低价判断是否触发，以近似标准海龟策略中的止损单/突破单执行方式。
- 买入价格包含正向滑点，卖出价格包含反向滑点。
- 入场与加仓每次只增加 1 个单位，最多 4 个单位。
- 止损和反向突破任一触发即卖出全部持仓。
- 基准净值采用首日收盘价买入并持有的口径。

## 注意事项

- 自动复权判断只作为提示，不能替代对数据源复权口径的确认。
- 海龟策略依赖趋势和波动，样本过短或波动过低时，信号可能稀疏或交易成本影响较大。
- 当前看板是单标的回测，组合级规则如高度相关市场 6 单位、同方向 12 单位会作为风险说明展示，不做跨市场持仓聚合。
