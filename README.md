# BA-AI-QUANT

BA工作坊-AI量化策略公益课-作业。

BA workshop weekly assignments and quantitative analysis artifacts.

## Structure

- `task1/`: SMIC A-share and H-share daily trading data dashboard.
- `task2/`: SMIC data diagnostics, technical indicator analysis, and an interactive indicator tool.
- `task3/`: Double moving-average strategy backtest dashboard, CLI engine, and sample data.

Future assignments should follow the same naming pattern: `task4/`, `task5/`, etc.

## Task 1

Open `task1/smic_dashboard.html` in a browser to view the local dashboard.

Included files:

- `task1/smic_a_daily_qfq.csv`: SMIC A-share front-adjusted daily trading data.
- `task1/smic_hk_daily_qfq.csv`: SMIC H-share front-adjusted daily trading data.
- `task1/qfq_metadata.json`: Adjustment factor source and generation metadata.
- `task1/smic_dashboard.html`: Interactive local HTML dashboard.
- `task1/smic_analysis_notebook.ipynb`: Notebook that reproduces the data loading, indicator calculation, and plotting workflow.
- `task1/build_dashboard.py`: Rebuilds the dashboard from the two front-adjusted CSV files.
- `task1/summary.json`: Summary statistics generated by the build script.

Notebook dependencies: `pandas`, `numpy`, and `matplotlib`.

## Task 2

Task 2 is organized by question:

- `task2/question1/`: Basic diagnostic analysis for SMIC A-share and H-share daily data.
- `task2/question3/`: Technical indicators and interactive HTML report.
- `task2/tool/`: Reusable local web tool for importing daily stock data and generating indicator charts.

### Question 1

Open `task2/question1/task2_question1_diagnostic_report.html` in a browser to view the diagnostic report.

Included files:

- `task2/question1/task2_question1_diagnostic_report.md`: Markdown diagnostic report.
- `task2/question1/task2_question1_diagnostic_report.html`: HTML diagnostic report.
- `task2/question1/task2_question1_diagnostic_analysis.ipynb`: Notebook workflow.
- `task2/question1/task2_question1_descriptive_stats.csv`: Combined descriptive statistics.
- `task2/question1/task2_question1_missing_summary.csv`: Missing-value summary.
- `task2/question1/task2_question1_ah_comparison.csv`: A-share vs H-share comparison table.
- `task2/question1/run_question1_analysis.py`: Rebuilds the question 1 outputs.

### Question 3

Open `task2/question3/task2_question3_technical_report.html` in a browser to view the interactive technical indicator report.

Included files:

- `task2/question3/task2_question3_technical_report.html`: Interactive HTML report with A-share/H-share switching.
- `task2/question3/task2_question3_technical_indicators.ipynb`: Notebook workflow.
- `task2/question3/task2_question3_indicators_a.csv`: A-share technical indicator output.
- `task2/question3/task2_question3_indicators_hk.csv`: H-share technical indicator output.
- `task2/question3/task2_question3_indicator_summary.csv`: Indicator summary table.
- `task2/question3/run_question3_analysis.py`: Rebuilds the question 3 outputs.

Question 3 indicators include RSI, MACD, Bollinger Bands, ATR, OBV, ADX, BIAS, and Z-score. The HTML report highlights key events such as MACD golden/death crosses, Bollinger Band breaks, RSI overbought/oversold points, and Z-score extremes.

### Indicator Tool

Open `task2/tool/stock_indicator_tool.html` to use the local web tool.

The tool supports:

- Importing local daily CSV files.
- Adjusting indicator parameters.
- Validating data completeness and reasonableness before chart generation.
- Generating the same style of SVG-based interactive indicator charts as question 3.
- Optional dynamic data fetching through the local Python backend.

Tool files:

- `task2/tool/stock_indicator_tool.html`: Main web tool.
- `task2/tool/stock_indicator_tool_server.py`: Optional local backend for dynamic data fetching.
- `task2/tool/stock_indicator_tool_config.example.json`: Backend/MCP configuration example.
- `task2/tool/stock_indicator_tool_readme.md`: Detailed usage and configuration guide.
- `task2/tool/smic_a_daily_qfq.csv`: Bundled SMIC A-share front-adjusted example CSV.
- `task2/tool/smic_hk_daily_qfq.csv`: Bundled SMIC H-share front-adjusted example CSV.

To start the optional backend:

```bash
python3 -B task2/tool/stock_indicator_tool_server.py
```

Then open:

```text
http://127.0.0.1:8765/stock_indicator_tool.html
```

## Task 3

Open `task3/double_ma_backtest_dashboard.html` in a browser to use the local double moving-average backtest dashboard.

Included files:

- `task3/README.md`: Detailed task 3 usage guide.
- `task3/task3_double_ma_backtest_dashboard_spec.md`: Confirmed dashboard specification.
- `task3/double_ma_backtest_dashboard.html`: Interactive local HTML dashboard.
- `task3/double_ma_backtest_engine.py`: CLI backtest engine for reproducible runs.
- `task3/double_ma_backtest_readme.md`: Additional usage notes.
- `task3/download_byd_a_data.py`: Downloads BYD A-share daily data and generates quality reports.
- `task3/sample_data/smic_a_daily_qfq.csv`: Bundled SMIC A-share front-adjusted example CSV.
- `task3/sample_data/byd_a_daily.csv`: BYD A-share raw daily data.
- `task3/sample_data/byd_a_daily_qfq.csv`: BYD A-share front-adjusted daily data.
- `task3/sample_data/byd_a_adj_factor.csv`: Implied BYD front-adjustment factor data.
- `task3/sample_data/byd_a_quality_report.md`: BYD data quality report.

The dashboard supports:

- Data quality checks and adjustment hints.
- Configurable backtest range, initial cash, MA windows, cost settings, and trend filter.
- Volume filter, position management, fixed/ATR stop loss, trailing stop, and drawdown control.
- Buy/sell/add markers with tooltip details and interactive legends.
- Price, moving-average, equity, and position charts.
- Trade list, signal log, final position, performance metrics, and HTML/PDF report export.

Run the CLI backtest with the bundled BYD data:

```bash
python3 -B task3/double_ma_backtest_engine.py task3/sample_data/byd_a_daily_qfq.csv --out-dir task3/exports_byd_default
```

Refresh BYD A-share data:

```bash
python3 -B task3/download_byd_a_data.py
```
