交易成本口径：

- 买入成本：手续费 + 滑点
- 卖出成本：手续费 + 印花税 + 滑点
- 每季度换仓一次，测试期第一个季度根据模型概率建仓。
- 若股票继续入选但权重变化，也按调仓差额近似计入交易成本。

止盈止损口径：

- 当前数据只有季度收益率，没有日内或日度价格路径。
- 因此止盈止损采用季度收益近似判断：
  - 若单只股票季度收益率高于 `TAKE_PROFIT`，该股票本季度收益按止盈收益计。
  - 若单只股票季度收益率低于 `STOP_LOSS`，该股票本季度收益按止损收益计。
- 看板中会明确说明这是季度数据下的近似口径，不等同于真实盘中触发。

### Step 3：构造应变量与衍生自变量

应变量：

```python

```

基础特征选择规则：

1. 排除 `Date`、`Code`、`Next_Ret`、`Y`
2. 其余可转换为数值型的列作为基础候选特征
3. 无法稳定转换为数值的字段不进入模型

衍生特征：

1. 估值类衍生：
   - PE、PB、PS、PCF 等指标的行业不可得时不做行业中性化，仅做截面分位数特征

2. 规模衍生：
   - `MV` 的对数值：`log_mv`
3. 历史收益衍生：
   - 按 `Code` 分组，使用历史 `Next_Ret` 滞后值计算

   - `ret_mean_4q`

4. RSI 过滤指标：

## 5. 三个模型训练步骤

### 5.1 逻辑回归

用途：

- 作为线性基准模型
- 输出可解释的标准化系数
- 使用 L1 和 L2 混合正则，降低过拟合并保留一定特征筛选能力

默认参数：

```python
LogisticRegression(
    penalty="elasticnet",
    solver="saga",
    l1_ratio=0.5,
    C=1.0,
    max_iter=5000,
    tol=0.0001,
    fit_intercept=True,
    class_weight="balanced",
    random_state=42
)
```

训练步骤：

1. 从训练集中取入模特征 `X_train` 和标签 `y_train`
2. 将无穷大替换为缺失值
3. 仅使用训练集计算每个特征的中位数
4. 用训练集中位数填充训练集和测试集
5. 仅使用训练集拟合 `StandardScaler`
6. 使用同一个 `StandardScaler` 转换训练集和测试集
7. 训练逻辑回归模型
8. 在测试集输出：

9. 计算分类评价指标

## 7. 交易策略设计

### 7.1 信号生成

每个模型在测试集中对每个股票和季度输出上涨概率：

```python
prob_up = model.predict_proba(X_test)[:, 1]
```

双阈值判定：

- 买入候选：`prob_up >= BUY_PROB_THRESHOLD`
- 明确规避：`prob_up <= SELL_PROB_THRESHOLD`
- 中间区间：观察，不主动买入

RSI 过滤：

- 若开启 RSI 过滤，仅保留 `RSI_MIN <= RSI <= RSI_MAX` 的股票。
- RSI 使用历史季度收益计算，避免未来数据。

趋势过滤：

- 若开启趋势过滤，仅保留过去 `TREND_WINDOW` 个季度累计收益 `>= TREND_MIN_RETURN` 的股票。
- 趋势指标使用历史季度收益计算，避免未来数据。

每季度选股：

1. 在测试集每个季度内，取该模型对应的所有股票预测概率
2. 先应用双阈值、RSI 过滤、趋势过滤
3. 按 `prob_up` 从高到低排序
4. 选择前 `TOP_N = 3` 只股票
5. 若不足 3 只，剩余资金持有现金

### 7.2 仓位分配

每个模型分别回测两种仓位方式。

等权：

- 入选股票数量为 `n`
- 每只股票权重为 `1 / n`
- 若 `n < 3`，未使用资金作为现金

概率加权：

- 仅对入选股票分配仓位
- 单只股票权重：

```python
weight_i = prob_up_i / sum(prob_up_selected)
```

- 若 `n < 3`，未使用资金作为现金

输出策略组合：

1. 逻辑回归 - 等权
2. 逻辑回归 - 概率加权
3. 决策树 - 等权
4. 决策树 - 概率加权
5. 随机森林 - 等权
6. 随机森林 - 概率加权

### 7.3 每季度收益率计算

单只股票季度收益：

```python
stock_ret = Next_Ret
```

止盈止损调整：

```python
adjusted_stock_ret = min(stock_ret, TAKE_PROFIT)
adjusted_stock_ret = max(adjusted_stock_ret, STOP_LOSS)
```

组合季度毛收益：

```python
gross_ret = sum(weight_i * adjusted_stock_ret_i)
```

交易成本：

- 建仓买入：按买入金额扣除手续费和滑点
- 季度调仓卖出：按卖出金额扣除手续费、印花税、滑点
- 权重变化产生的买卖差额也计入交易成本

组合季度净收益：

```python
net_ret = gross_ret - turnover_cost
```

资金曲线：

```python
capital_t = capital_{t-1} * (1 + net_ret)
```

每个季度输出：

- 日期
- 模型
- 仓位方式
- 入选股票代码
- 各股票预测概率
- 各股票权重
- 各股票原始季度收益
- 各股票止盈止损后收益
- 组合季度毛收益
- 换手率
- 交易成本
- 组合季度净收益
- 期末资金

## 8. 基准策略

为便于对比，增加两个基准：

1. 测试集全市场等权基准
   - 每季度持有测试集中全部股票等权组合
2. 测试集随机选 3 只股票基准
   - 每季度随机选择 3 只股票等权持有
   - 使用 `RANDOM_STATE` 固定随机结果，保证可复现

基准同样扣除交易成本。
